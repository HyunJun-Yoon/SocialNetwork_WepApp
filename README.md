# Murakano - Social Network Platform

A full-stack social networking application built with the MERN stack (MongoDB, Express, React, Node.js). Users can create profiles, share posts, comment on posts, and connect with other developers.

## Features

- **User Authentication**: Secure registration and login with JWT tokens
- **User Profiles**: Create and edit user profiles with education and work experience
- **Posts & Comments**: Share posts and engage with community through comments
- **Profile Discovery**: Browse and view other users' profiles
- **Dashboard**: Personalized user dashboard with profile management
- **Responsive Design**: Bootstrap-powered responsive UI for all devices
- **State Management**: Redux for centralized state management

## Tech Stack

### Frontend
- **React.js** - UI library
- **Redux** - State management
- **React Router** - Client-side routing
- **Bootstrap 5** - CSS framework
- **Axios** - HTTP client

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **JWT** - Authentication
- **Bcryptjs** - Password hashing
